# Project Instructions for Claude Code

Use `/secure-review` for authorized defensive source code review. Default to detection-only. Do not expose secrets. Do not edit source code unless explicitly asked.

For one application at a time:
1. profile app,
2. map attack surface,
3. map roles and permissions,
4. select playbooks,
5. run safe deterministic tools,
6. validate findings,
7. report with evidence.
