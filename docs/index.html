claude

cp -R packages/codex/secure-review-codex/.agents /path/to/app/.agents

codex

Workflow

  1. Confirm scope and authorization.
  2. Profile the application, runtime, framework, database, and deployment model.
  3. Map routes, APIs, jobs, webhooks, queues, admin paths, mobile paths, and file handlers.
  4. Map authentication, roles, permissions, tenant boundaries, and object ownership.
  5. Select playbooks and safe deterministic scans.
  6. Collect candidate findings, then apply the validation gate and false-positive review.
  7. Report only evidence-backed findings with safe retest steps and remediation.
A final finding needs reachable code evidence, attacker-controlled input or missing control, affected role or tenant where relevant, realistic impact, and a safe validation path.
